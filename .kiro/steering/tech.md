# Technology Stack

## Framework
- **Astro**: Static site generator with component islands architecture
- Supports multiple UI frameworks (React, Vue, Svelte, etc.) as needed for interactive components

## Build System
- Node.js and npm/pnpm/yarn package manager
- Astro's built-in build system

## Common Commands
```bash
# Development
npm run dev          # Start dev server (typically http://localhost:4321)
npm run build        # Build for production
npm run preview      # Preview production build locally

# Alternative with pnpm
pnpm dev
pnpm build
pnpm preview
```

## Key Libraries & Tools
- Data visualization libraries (to be determined based on requirements)
- Markdown/MDX for content authoring
- TypeScript for type safety (if configured)

## Deployment
Static site output can be deployed to any static hosting service (Netlify, Vercel, GitHub Pages, etc.)
